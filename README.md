# squids.co.za

Official website for **Squids** — Africa's Social Platform.

Built with pure HTML, CSS, and JavaScript. No frameworks. No build step. Deploys directly via GitHub Pages.

## 🚀 Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Set custom domain to `squids.co.za`
5. Add a CNAME record at your DNS provider: `squids.co.za → your-username.github.io`
